What is HTML?

is the "language" used to tell a computer what you want it to do. html stands for hypertext markup language. html documents are comprised of elements and tags. Tags are the specific instructions that tell the elements what to do and are always in < >. tags always have an open comand and a closing comand. Void tags do not require an end tag because it does not contain an element. Elements refer to everything within a set of "opening" and "closng" tags.
